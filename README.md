rs_bridge

QBCore/QBox compatibility bridge for FiveM resources — built for Reality Sucks RP scripts.

![Framework](https://img.shields.io/badge/Framework-QBCore%20%7C%20QBox-blue)
![Version](https://img.shields.io/badge/Version-1.0.0-green)

---

Features

- Auto-detects QBCore or QBox
- One resource works on both frameworks
- Reduces duplicate scripting work
- Clean wrapper system for exports
- Built for Reality Sucks RP ecosystem

---

Purpose

`rs_bridge` allows you to build once and run everywhere.

Instead of maintaining separate versions for QBCore and QBox, this bridge handles framework detection and routing automatically.

---

Supported Frameworks

- QBCore  
- QBox / qbx_core  

---
